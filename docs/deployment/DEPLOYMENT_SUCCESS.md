## 📞 **SUPPORT & TROUBLESHOOTING:**

### **Login Issues:**
1. Try "Forgot Password" link
2. Check email is correct
3. Check caps lock is off
4. Clear browser cache

### **Can't See Dashboard:**
1. Verify role is set correctly (Admin Panel → Manage Users)
2. Check "active" status is enabled
3. Sign out and sign in again

### **Time Entry Not Saving:**
1. Check internet connection
2. Check browser console for errors (F12)
3. Verify yesterday's entry is complete
