## TimeTrack Week-1 Operations Checklist

### Daily (5–10 minutes)
- **Team Dashboard**: spot-check “Incomplete” entries before cutoff.
- **Corrections**: use **Correct Entry** (admin) with a clear admin note for any fixes.
- **Suspicious signals**: review any flagged/suspicious items if you use Pattern Metrics.

### Payroll cycle (biweekly)
- **Run Payroll Report** for the pay period.
- **Sanity checks**:
  - Compare 2–3 employees to their expected schedule/typical hours.
  - Verify any overtime spikes.
  - Confirm lunch compliance where required.

### User lifecycle
- **New hire**: create user + role; confirm they can log in.
- **Role changes**: update user role (employee ↔ manager).
- **Terminations/leavers**: set user to **Inactive** immediately.

### If something looks wrong
- **Audit Viewer**: review changes and submission timestamps.
- **Pattern Metrics**: look for repeated identical patterns or unusual edits.
